```typescript
export function renderProportionalFrame(
  ctx: CanvasRenderingContext2D,
  frame: AsciiFrame,
  lineHeight: number,
  fgColor: string,
  bgColor: string,
  colorMode: ColorMode,
): void
```

Implementation:
1. Clear canvas with bgColor fillRect
2. Set textBaseline = 'top'
3. For each row in frame.cells:
   a. Build rowText from cells
   b. Call `prepareWithSegments(rowText, ctx.font, { whiteSpace: 'pre-wrap' })`
   c. Call `layoutWithLines(prepared, ctx.canvas.width, lineHeight)` (used for validation, not for per-char positioning)
   d. Iterate columns: use `ctx.measureText(cell.char).width` to accumulate x position
   e. If x >= canvas.width, break (D-02 clip)
   f. Set fillStyle based on colorMode (same logic as renderGridFrame)
   g. Call ctx.fillText(cell.char, x, y) where y = row * lineHeight
   h. Skip space chars (advance x but do not fillText, matching grid behavior)

Import `prepareWithSegments` and `layoutWithLines` from `'@chenglou/pretext'` at top of renderer.ts.

Note: `renderProportionalFrame` does NOT take `charWidth` parameter — character widths are variable, measured per-character.

**Step 2 — Create packages/player/src/typewriter.ts:**

```typescript
export class TypewriterReveal {
  private _charDelay: number
  private _timeoutId: ReturnType<typeof setTimeout> | null = null
  private _cancelled: boolean = false

  constructor(charDelay: number = 30) {
    this._charDelay = charDelay
  }

  get charDelay(): number { return this._charDelay }

  reveal(
    totalChars: number,
    onChar: (revealCount: number, charIndex: number, timestamp: number) => void,
    onComplete: (timestamps: number[]) => void,
    startTime: number = 0,
  ): void {
    this._cancelled = false
    const timestamps: number[] = []
    let i = 0

    const step = () => {
      if (this._cancelled || i >= totalChars) {
        if (!this._cancelled) onComplete(timestamps)
        return
      }
      const t = startTime + i * this._charDelay
      timestamps.push(t)
      onChar(i + 1, i, t)
      i++
      this._timeoutId = setTimeout(step, this._charDelay)
    }
    step()
  }

  cancel(): void {
    this._cancelled = true
    if (this._timeoutId !== null) {
      clearTimeout(this._timeoutId)
      this._timeoutId = null
    }
  }
}
```

```typescript
export function renderTypewriterFrame(
  ctx: CanvasRenderingContext2D,

  charWidth: number,
  lineHeight: number,
  fgColor: string,
  bgColor: string,
  colorMode: ColorMode,
  revealCount: number,
  showCursor?: boolean,
): void
```

Implementation:
1. Clear canvas with bgColor fillRect
2. Set textBaseline = 'top'
3. Flatten frame.cells into a linear index (row-major)
4. Iterate cells up to `revealCount` — for each, compute (col, row) position and draw with fillText at (col * charWidth, row * lineHeight)
5. Color logic same as renderGridFrame
6. If `showCursor` is true and revealCount < total chars, draw a block cursor ('|') at the next position using fgColor
7. Characters beyond revealCount are not rendered (partial reveal effect)

This function uses grid-mode fixed charWidth positioning. Proportional-typewriter combination is out of v1 scope (typewriter uses grid layout).
