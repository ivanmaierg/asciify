**Step 1 — Create packages/player/tests/snapshot.test.ts (TEST-04):**

This file captures deterministic canvas call sequences using vitest inline snapshots. Use a mock ctx that records all draw calls as strings.

Create a `makeRecordingCtx(width, height)` helper:
```typescript
function makeRecordingCtx(width = 200, height = 100) {
  const calls: string[] = []
  const canvas = { width, height }
  const ctx = {
    canvas,
    fillStyle: '',
    textBaseline: '',
    font: '14px monospace',
    fillRect: (x: number, y: number, w: number, h: number) => {
      calls.push(`fillRect(${x},${y},${w},${h})`)
    },
    fillText: (t: string, x: number, y: number) => {
      calls.push(`fillText(${JSON.stringify(t)},${x},${y})`)
    },
    measureText: (t: string) => ({ width: t.length * 8 }),
    clearRect: vi.fn(),
  }
  return { ctx: ctx as unknown as CanvasRenderingContext2D, calls }
}
```

Write snapshot tests for:

1. **Grid mode snapshot** — render a 2x2 frame, capture call sequence, use `expect(calls).toMatchInlineSnapshot(...)`. Let vitest fill in the inline snapshot on first run (run with `--update`).

2. **Proportional mode snapshot** — render a 1-row frame with 3 chars ('A', 'B', 'C'), capture the prefix-sum positioning. Since measureText returns `text.length * 8`, positions should be 0, 8, 16.

3. **Typewriter mode snapshot** — render a 1-row 3-char frame with `revealCount=2`, verify only 2 chars drawn plus cursor.

4. **Typewriter mode revealCount=0 snapshot** — verify only fillRect (background clear), no fillText calls.

5. **Proportional mode colored snapshot** — verify fillStyle changes per cell RGB.

Use `import { renderGridFrame, renderProportionalFrame, renderTypewriterFrame } from '../src/renderer'` directly.

For proportional mode, the mock ctx must also handle the `prepareWithSegments`/`layoutWithLines` calls. Since those are internal to pretext and may not work with a mock ctx, mock the `@chenglou/pretext` module:
```typescript
vi.mock('@chenglou/pretext', () => ({
  prepareWithSegments: vi.fn(() => ({ segments: [] })),
  layoutWithLines: vi.fn(() => ({ lines: [{ text: '', width: 0 }] })),
}))
```

This ensures the snapshot captures the drawing calls without depending on pretext internals.
