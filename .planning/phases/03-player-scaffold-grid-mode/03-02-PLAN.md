    **playback.ts** (per D-13, D-14, PLR-04, PLR-05, PLR-06, PLR-12):
    Create `packages/player/src/playback.ts` exporting `PlaybackController` class:

    ```typescript
    export interface PlaybackCallbacks {
      onFrame: (frameIndex: number) => void
      onTimeUpdate: (currentTime: number) => void
      onEnded: () => void
    }

    export class PlaybackController {
      constructor(totalFrames: number, fps: number, loop: LoopMode, callbacks: PlaybackCallbacks)
      play(): void
      pause(): void
      seekTo(seconds: number): void
      seekToFrame(index: number): void
      destroy(): void
      get isPlaying(): boolean
      get currentFrameIndex(): number
      get currentTime(): number
      get duration(): number
      set fps(value: number)
      set loop(value: LoopMode)
    }
    ```

    Implementation details:
    - `_tick(timestamp: number)`: rAF callback. Calculate `elapsed = timestamp - lastFrameTime`. If `elapsed >= 1000 / fps`, call `callbacks.onFrame(currentFrameIndex)`, then `_advance()`. Use drift correction: `lastFrameTime = timestamp - (elapsed % frameDuration)`.
    - `_advance()`: increment frameIndex. If past last frame, check loop mode. 'once' -> pause + onEnded. 'forever' -> reset to 0. number -> decrement, if 0 then pause + onEnded, else reset to 0.
    - `play()`: if not playing, set `_isPlaying = true`, call `requestAnimationFrame(this._tick)`.
    - `pause()`: set `_isPlaying = false`, `cancelAnimationFrame(_rafId)`.
    - Fire `onTimeUpdate` callback on each frame advance (every rendered frame, not throttled — per Claude's discretion).
    - `destroy()`: pause + null out callbacks.

    **_init() method:**
    1. Auto-detect format: if `data.frames.length > 0 && typeof data.frames[0] === 'string' || Array.isArray(data.frames[0])` then compact format (D-08)
    2. If compact: eagerly decompress all frames. Walk `EncodedFrame[]`:
       - String frame (keyframe): `rleDecode(frame)` -> full text. Build `AsciiFrame` with text only, empty cells (compact doesn't have cells — monochrome rendering only).
       - Array frame (delta): apply patches to previous frame's text. `DeltaPatch = [position, changedChars][]` — splice changes into previous text string.
    3. Store decompressed `AsciiFrame[]` as `this._frames`
    4. Font load: `await document.fonts.load(this._font)` where `this._font` is the resolved font string (PLR-11, MODE-05)
    5. Measure char width: `ctx.font = this._font; const charWidth = ctx.measureText('M').width` (grid mode, per research recommendation)
    6. Calculate lineHeight: `const fontSize = parseFloat(this._font); const lineHeight = fontSize * 1.2` (standard line height)
    7. Store charWidth, lineHeight for rendering
    8. Create PlaybackController with frame count, fps, loop mode, and callbacks that call renderGridFrame and dispatch events
